## 2026-05-06 - Preventing O(N) re-renders in mapped lists with React.memo
**Learning:** When using `React.memo` on list items, conditionally passing `null` instead of active state IDs (like `highlightedId` or `mutatingId`) to items that do NOT match the ID prevents O(N) re-renders across the entire list. However, applying this optimization to nested structures (like a comment thread with replies) will cause functional regressions because nested child nodes will never receive the matching ID from their parent if the parent filters it out first.
**Action:** Apply this optimization safely to flat lists only. For nested structures, either pass the ID down regardless, check if any descendants contain the target ID before bailing, or use React Context to subscribe individual items directly to the active IDs.
