## 후기 어뷰징 방지 정책 메모

질문: "스탬프를 찍은 그 날만 후기를 올리게 제한하면 어뷰징 방지에 도움이 되는가?"

판단:
- 일부 도움은 된다.
- 하지만 "그 날만"은 너무 빡세다. 현장에서는 구경만 하고, 이동 후 사진 정리나 문구 정리는 나중에 하는 사용자가 많기 때문이다.
- 따라서 어뷰징 방지 목적이라면 `당일 한정`보다 `현장성 + 횟수 제한` 조합이 더 낫다.

권장 정책:
- 해당 장소 스탬프를 적립한 사용자만 후기 작성 가능
- 후기 작성 가능 시간: 스탬프 적립 시각부터 24시간 이내
- 장소별 1일 1회 작성 제한
- 기존 후기 수정은 허용, 신규 중복 작성은 제한
- 관리자 숨김/차단 기능 유지

현재 구현 상태:
- 아직 미적용
- 지금은 로그인한 사용자가 장소를 선택하면 후기 작성이 가능하다
